## Installation
Installation is simple.

After downloading the ZIP package, extract it and move the files to the "lockscreen" folder which is inside the "Kustom" folder which by default is on the internal storage of the target device.

After that, it's a matter of opening the KLCK application and loading the presets from the Library section.

## Settings location

Preset settings are located in the "Globals" tab of the editor further in the KLCK app.
